blacklist-follow-type

Before using, please log in to your Kakuyomu account. Then click the red or ignore button to set the default block type. If the blacklist or following-list is updated, the search page should also be refreshed (not navigated back).

You can set individual tag types for each followed user at https://kakuyomu.jp/users/*/following_users.
Click "visible". Green means the user tag will be green on the search page, while red means the user will be hidden.
Finally, reload the search page.(not rollback)
This plugin works on https://kakuyomu.jp/search*
https://kakuyomu.jp/tags/*
This plugin requires refreshing the search page twice when it's updated or loaded for the first time. The first refresh is used to fetch data, and it will only work after the second refresh.
Sometimes, changes won't appear immediately after refreshing the page when the list is updated because it's waiting to retrieve new data. The changes will usually appear after the next refresh.
